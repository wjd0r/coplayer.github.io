Java IP address 사용하기

Nov 1 2019 Technology

1. 인지

각각 REST api 개발을 하다보면 개발환경의 IP 주소를 동적으로 할당하여 사용해야 하는 경우가 대부분입니다. 각 개발자들의 개발환경에 맞는 아이피가 할당되도록 프로그램을 수정하던 도중에 사용한 클래스에서 이상한 아이피가 할당되는 것을 확인하게 되었습니다.

2. 분석

InetAddress 내부를 살펴보면 호스트네임이 “localhost” 일때 정확한 IP를 리턴 하도록 구현되어 있습니다. 유닉스와 리눅스 계열에서 네임이 사용자에 따라 동적으로 바뀌면서 IP가 제대로 할당되지 않는 것이 었습니다.

3. 해결

분석 이후 해결책을 구글링 하였습니다.

위의 코드로 IP를 호출하면 정상적으로 로컬의 주소를 가져올 수 있습니다.

4. 추가

IPv4 와 IPv6 할당의 차이인지는 확인이 필요합니다.

JSON 이란

Nov 1 2019 Old

JSON ?

  • JSON(제이슨, JavaScript Object Notation)은 속성-값 쌍으로 이루어진 데이터 오브젝트를 전달하기 위해 인간이 읽을 수 있는 텍스트를 사용하는 개방형 표준 포맷이다. (wiki 참조)
    프로그래밍 언어와 플랫폼에 독립적이므로, 서로 다른 시스템간에 객체를 교환하기에 좋다.

  • 숫자 : C나 자바에서의 8진수와 16진수를 표현하는 방법은 지원되지 않는다.

  • 문자열(String) : “”로 묶어서 사용

  • 배열(Array) : [] 로 묶어서 {} 선언하여 사용

  • 객체(Object) : {} 묶어서 사용({“name2”: 50, “name3”: “값3”, “name1”: true})

  • link Page (http://www.json.org/) - 각 언어별 JSON 사용법

Json {“@nil” : “true”}

t.industry instanceof Object // result is: true
t.industry instanceof Array // result is: false

개발에 공부에 도움이 되는 사이트 모아보기

Nov 1 2019 Old

개발에 관련되서 강의나 사이트가 수도 없이 많은 만큼
옥석을 가리는 것도 그만큼 중요한 일이 되었다.

개인적으로 좋은 사이트를 모아놓자.

- 생활코딩

웹 개발관련해서 공부를 시작한 국내 사람이라면 한번쯤은 들어가본 사이트가 아닐까? 개인적으로는 web1 강의를 보고 개발이란? 에 대해서 다시 생각해보게 되었다.

- 프로그래머스

동영상 강의 및 알고리즘연습 환경을 제공한다. vscode와 거의 동일한 코딩 환경을 지원 문제수나 종류가 많아지면 알고리즘 연습하기 좋은 사이트가 될것 같다.

- 엔지니어대한민국 YouTube채널

웹 서핑을 하다가 우연히 알게된 유툽 영상의 편집이 강의지만 속도감 있고 기본적인 자료구조 설명부터 기초적인 알고리즘 구조에 대해서 이론을 설명하고 코드로 보여준다.
직접 구현하시는 코드가 자바로 구성되어 있고 객체지향성이 눈에 띈다.

- Wikidocs

온라인 책 제작 공유사이트(개발관련 서적이 다수) 점프투 파이썬, 점프투 자바 등등

- 인프런

양질의 무료강의와 유료강의 등 (Recursion을 이해하기 좋은 강의)

- BOJ

기초부터 고급까지 방대한 문제, 커뮤니티가 활성화 되어 있어 질문하기 좋은 곳이다.
